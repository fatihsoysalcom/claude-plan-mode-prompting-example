# Claude Plan Mode Prompting Example

This Python script simulates Claude's 'plan mode' by first asking an AI to generate a plan for a task, and then guiding the AI through executing each step of that plan. It illustrates a 'router' rather than a 'forcer' approach to AI interaction.

## Language

`python`

## How to Run

Run this Python script directly: `python plan_mode_demo.py`
